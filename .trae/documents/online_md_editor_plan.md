# 在线Markdown编辑器 - 实现计划

## 项目概述
创建一个功能完整的在线Markdown编辑器，支持实时预览、语法高亮、常用编辑工具和导出功能。

## 技术栈
- **前端框架**: Vue 3 + Vite
- **Markdown解析**: marked
- **代码高亮**: highlight.js
- **CSS预处理器**: SCSS

│   │   └── export.js        # 导出功能工具
│   ├── styles/

├── public/                  # 静态资源
